Assignment: Milti-channel controller practice

by Yudi Zhang

Project Description:

I want the screen to have a star, buttons to control the random change of the background color, and a potentiometer to control the size of the star. And in this way to have a preliminary experience with multi-controller. Materials used: ESP32 feather, dupont wire, resistors, button, Potentiometer.

Coding Documentation:

first step - assembling

I connect all the materials mentioned above to a breadboard and ready to coding.

second step - arduino coding

First I defined the potentiometer and button pins in arduino and then set up the framework for communication.

thrid step - processing coding

I created a function in Processing to draw the star and make its size vary according to the potentiometer value received from the ESP32. Also randomly change the background color when a button press is received.
