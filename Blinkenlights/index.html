Assignment: Blinkenlights

Project Name: Morse Code - ZYD

by Yudi Zhang

Project Description:

I used my led lights to create three rhythmic patterns based on Morse code and the message it hides is Z-Y-D. Materials used: ESP32 feather, dupont wire, resistors, led light.

Coding Documentation:

first step - assembling

I connect all the materials mentioned above to a breadboard and ready to coding.

second step - arduino coding

I used in arduino, first to set up my led lights with HIGH and LOW; then designed my blinking frequency to be long and short, long and short, long and short.

thrid step - code testing

To ensure results, I first tested to use a button to control the frequencies of all three lights in turn. The test video is also attached.
