Assignment: Time and oscillation

by Yudi Zhang

Project Description:

I referenced Wassily Kandinsky's work to create an animation with at least two moving shapes - one that moves in a repetitive motion, and another that performs the movement after a predetermined time interval.

Coding Documentation:

Project setup

1.In the `setup()` function, create a 400x400 pixel canvas using `createCanvas(400, 400)` and set the frame rate of the animation to 60 frames per second using `frameRate(60)`. In the `draw()` function, a background is set with `background(255, 250, 200)` at the beginning of each frame.

Add dynamic shapes

2.Use `ellipse()` to draw a yellow circle moving up and down in the upper left corner. The vertical position of the circle changes over time, using the sine function

3.Draws a white triangle in the lower left corner that moves left and right using `triangle()`. The horizontal position of the triangle also uses the sine function to achieve circular motion.

4.Draws a rectangle that simulates a pendulum using `rect()`. Its swing is time-based, starting every 5 seconds and lasting 3 seconds.

5.Use `rect()` to draw a square that simulates a rotating planet. Its rotation is also time-based control, starting every 7 seconds and lasting 4 seconds.

6.The red circle is drawn using `ellipse()` and its position is based on the sine and cosine functions, simulating oscillating motion.

7.The blue circle appears at specific time intervals and its movement is based on the `noise()` function, creating an irregular movement effect.
