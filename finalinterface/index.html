Assignment: Finish your interface

by Yudi Zhang

Project Description:

Like the reference I found last week, based on ml5j's face-api, I'd like to make a webcam interactive filter game using p5js. When it starts running, the webcam recognizes the user's face and lets the user's face control a red circle; the circle will run forward at the bottom of the screen while yellow dots randomly appear, and when the user controls the red circle and touches the yellow dots, it disappears and displays a score on the screen.

Coding Documentation:

Project setup

1.First, create an HTML file and bring in the p5.js and ml5.js libraries.

Initialize Webcam and Facial Recognition

2.In the sketch.js file, setup() function of p5.js to initialize webcam and Face API of ml5.js.

Create red circle and yellow dot

3.In sketch.js, define a red circle and a set of randomly appearing dots.

Control the red circle

4.Use facial recognition data to control the movement of the red circle.

PHandling interactions and collision detection

5.In the draw() function, draw the red circle and the star and detect collisions between them.
